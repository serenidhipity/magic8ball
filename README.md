# ask the ball 🎱

A minimalist, interactive Magic 8-Ball web application designed with a focus on clean aesthetics and expressive typography.

## Features
- **Clean Aesthetic:** High-contrast, minimalist design with a focus on clarity.
- **Expressive Typography:** Utilizes the unique `Bitcount Grid Single` typeface for a creative, artistic feel.
- **Classic Responses:** Includes all 20 original Magic 8-Ball answers.
- **Interactive:** Smooth CSS animations for a physical "shake" effect when triggered.

## Typography
- **Heading & Controls:** [Bitcount Grid Single](https://fonts.google.com/specimen/Bitcount+Grid+Single)
- **Ball Answers:** [Unica One](https://fonts.google.com/specimen/Unica+One)

## Tech Stack
- **Frontend:** Vanilla HTML5, CSS3, and JavaScript.
- **Fonts:** Google Fonts API.
